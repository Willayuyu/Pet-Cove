# Vue E-commerce

> An E-commerce platform application made with bootstrap4
> and Vue/Vuex/Vue-router.

## Development/Producation Setup

``` bash
# Open the project folder
cd Front-end && npm install

npm run serve

# FOR BUILD: 
npm run build

# FOR AXIOS
npm install axios

# FOR CROS (Cross-Origin Resource Sharing)
npm install cors


```
